Welcome Fire Oracle

This is my attempt to combat the unprecedented wildfires occurring every year in the United States.
With my recent interest in Machine Learning, and the unfortunate news of Wildfires occurring
in United States, I figured this would be a great opportunity to practice ML and learn to create
a website. Though the main focus of this project at this time is building a website using javascript,
I hope to create a minimally viable prototype that I can continue to work on and refine after this
semester. With that being said the goal is to create a model that can predict at least
one wildfire in the United States, and to create a mostly functional website.

Want to help join me in GitHub!
